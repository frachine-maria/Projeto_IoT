# Projeto_IoT
Projeto IoT - forno de cerâmica. 

Descrição: O que o projeto faz?
    O projeto tem como objetivo criar um sistema de monitoramento e segurança para ambientes de trabalho em empresas de cerâmica, onde os funcionários ficam próximos a fornos e chapas de alta temperatura.

Hardware: Lista de materiais utilizados (Sensores, Atuadores, Placa).
    Raspberry Pi Pico – Microcontrolador responsável por processar os dados e controlar o sistema.
    DHT22 – Sensor digital usado para medir temperatura e umidade do ambiente próximo ao forno.
    HC-SR04 – Sensor ultrassônico utilizado para detectar a aproximação de pessoas ou objetos.
    Buzzer – Atuador responsável por emitir um alerta sonoro em situações de risco.
    LEDs (verde, amarelo e vermelho) – Indicadores visuais que representam os níveis de segurança do ambiente.
    Display LCD – Exibe informações como temperatura, umidade e mensagens de alerta.
    Protoboard e Jumpers – Usados para montar e conectar todo o circuito.
